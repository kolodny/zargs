Zargs
====

[![Build Status](https://travis-ci.org/kolodny/zargs.svg)](https://travis-ci.org/kolodny/zargs)
[![NPM version](https://badge.fury.io/js/zargs.svg)](http://badge.fury.io/js/zargs)

Usage:
===

```js
require('zargs')([
    {
        key: 'f',
        alias: 'file',
        demand: true,
        describe: 'file to open'
    }
])
```
